Test Servlet

Această aplicație afișează informații despre platforma Java Servlet unde rulează.

  • La URL ./servletapiversion puteți verifica numărul de versiune API al platformei servlet
  • La URL ./servletplatform puteți verifica identificatorul platformei servlet (Tomcat/Jetty/Liberty/JBoss, etc)

Aplicația a fost testată cu platformele:

  • Tomcat 7
  • Tomcat 8
